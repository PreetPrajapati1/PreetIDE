# 🚀 PreetIDE – Web-Based Code Editor

PreetIDE is a modern **web-based IDE** that allows users to write and execute code directly in the browser.
Built using the powerful Monaco Editor (the same editor behind VS Code) and a Node.js backend.

---

## ✨ Features

* 🧠 Smart Code Editor (Monaco Editor)
* 🎨 Syntax Highlighting
* ▶️ Run JavaScript or Python Code in Real-Time
* 📦 Backend Execution using Node.js
* 💻 Clean and Responsive UI
* ⚡ Fast and Lightweight

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript
* Monaco Editor

**Backend**

* Node.js
* Express.js

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/preet-ide.git
cd preet-ide
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the server

```bash
node server/server.js
```

### 4️⃣ Open in browser

```
http://localhost:3000
```
